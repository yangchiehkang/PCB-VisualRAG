Introduction

This document describes the NCV7535 EVB board for the
ON Semiconductor NCV7535 SPI controlled H-bridge and
Dual-Half Bridge pre-driver. The functionality and major parameters
can be evaluated with the NCV7535 EVB board.

The NCV7535 is a monolithic SPI controlled chip with enhanced
feature set useful in automotive systems. Besides the SPI bus
interface, the IC features an H-bridge pre-driver to control a
DC-motor. This allows a highly integrated solution.
